Counting Genres & Supergenres

With the excel sheet of the Top 2000 that NPO Radio 2 provides it was easy enough to create a small interactieve visualisation of the songs, artists and release years. But I thought one more dimension would add a lot of value: the musical genre. To get the genre of each song I used [Last.fm](http://Last.fm). I wrote a pretty small loop and a few functions which were able to create a hyperlink for each artist that would lead to their page on this website, which lists the genres of their music! With some usage of the BeautifulSoup package I was able to extract this data from the html of the page.
